### Backend development

1) create git repo
    git init

2) add .gitignore file

3) add .env file for environmental variables & read data from .env with "dotenv" module(npm install dotenv)

4) generate package.json (npm init -y) (change the main to server.js and type to module)

5) create express app

6) create server.js

7) connect to database: install moongoose

8) install nodemon for automatic start of server instead of manual starting

9) Add middlewares(body parser,error handler)

10) Design schemas and create models 

11) Design REST APIs for all resources and connect it in server.js
